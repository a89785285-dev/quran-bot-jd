# 🌙 بوت القرآن الكريم

بوت تيليجرام لنشر آيات قرآنية تلقائياً مع دعم الجدولة الزمنية والقنوات.

## ✨ المميزات

- 📖 إرسال آيات قرآنية عشوائية تلقائياً
- ⏰ جدولة مخصصة للإرسال (كل 5 دقائق أو أكثر)
- 🎯 البحث عن آيات من سورة محددة
- 📢 دعم إرسال الآيات في القنوات
- 🌍 استخدام API القرآن السحابية
- 🇸🇦 دعم كامل للغة العربية

## 🚀 البدء السريع

### المتطلبات

- Python 3.8+
- توكن بوت تيليجرام

### التثبيت

```bash
# استنسخ المستودع
git clone https://github.com/a89785285-dev/quran-bot-jd.git
cd quran-bot-jd

# ثبت المكتبات
pip install -r requirements.txt
```

### الإعداد

```bash
# ضع التوكن في متغير البيئة
export TELEGRAM_BOT_TOKEN='your_token_here'

# أو أنشئ ملف .env
cp .env.example .env
# ثم عدّل الملف وضع التوكن
```

### التشغيل

```bash
python bot.py
```

## 📋 الأوامر المتاحة

### للمحادثات الخاصة والمجموعات

| الأمر | الوصف |
|------|-------|
| `/start` | بدء البوت وعرض المساعدة |
| `/help` | عرض جميع الأوامر المتاحة |
| `/startquran` | بدء إرسال آية كل 5 دقائق |
| `/schedule <دقائق>` | تخصيص الفترة الزمنية (مثال: `/schedule 10`) |
| `/stopquran` | إيقاف الإرسال |
| `/verse` | جلب آية عشوائية الآن |
| `/surah <اسم أو رقم>` | آية من سورة محددة (مثال: `/surah البقرة` أو `/surah 2`) |

### للقنوات

| الأمر | الوصف |
|------|-------|
| `/setchannel <@قناة> <دقائق>` | بدء الإرسال الآلي في قناة (مثال: `/setchannel @mychannel 30`) |
| `/stopchannel <@قناة>` | إيقاف الإرسال في قناة |

## 📖 أمثلة الاستخدام

### إرسال آيات كل 5 دقائق
```
/startquran
```

### إرسال آيات كل 10 دقائق
```
/schedule 10
```

### الحصول على آية من سورة البقرة
```
/surah البقرة
```

### إرسال آيات في قناة كل 30 دقيقة
```
/setchannel @my_quran_channel 30
```

## 📚 API المستخدمة

البوت يستخدم [Quran Cloud API](https://alquran.cloud/) وهي API مجانية توفر:
- جميع آيات القرآن الكريم
- معلومات السور
- ترجمات متعددة

## ⚙️ الإعدادات المتقدمة

### تغيير مصدر البيانات

يمكنك تعديل `API_URL` في `bot.py` لاستخدام API مختلفة:

```python
API_URL = 'https://api.alquran.cloud/v1'
```

## 🐛 استكشاف الأخطاء

### البوت لا يستجيب
- تأكد من صحة التوكن
- تحقق من الاتصال بالإنترنت
- تأكد من تثبيت جميع المكتبات: `pip install -r requirements.txt`

### خطأ في جلب الآيات
- تحقق من الاتصال بـ API: `https://api.alquran.cloud/v1/quran`
- قد تكون هناك مشكلة مؤقتة في الخادم
